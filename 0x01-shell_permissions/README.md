	Shell Permission
su	- Switches the current user
whoami	- Prints the effective username of the current user
groups	- Prints all the groups the current user is
chown	- Changes the owner of a file
touch	- Creates an empty file
chmod	- Adds execute permission to the owner of the file hello
chmod	- Adds execute permission to the owner and the group owner, and read permission to other users, to the file hello
